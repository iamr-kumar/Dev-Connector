# Dev-Connector
A social media webiste for developers

### Link for live demo
https://dev-connector-rm101.herokuapp.com/

### Installation
1. Clone repository
```sh 
git clone git@github.com:iamr-kumar/Dev-Connector.git 
```
2. Install dependencies
```sh 
npm install
```
3. Start server
```sh 
node server.js
```
